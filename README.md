# email-data
Command Line Python app to pull/enter/and see email data.

Usage: Python3 main.py 'csv-files/<filename>' [optional <campaign_initial>] flag(s)

command line flags:
 - '-t / --top' = will show you the best performing email of the week.
 - '-w / --worst' = shows the worst performing email of the week.
 - '-b / --best(n)' = shows the top (n) emails of the week.
 - '-a / --all' = shows all emails in sequence with their stats.
